# Simple 2D Platformer

A basic 2D platformer game built with Pygame.

## How to Run

1. Make sure you have Python installed.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the game: `python main.py`

## Controls

- Left Arrow: Move left
- Right Arrow: Move right
- Space: Jump

## Features

- Player (princess sprite) movement and jumping with double jump
- Gravity
- Platform collision
- 3 levels with increasing difficulty
- Collectible coins (10 points each)
- Obstacles that reset the player and deduct a small penalty
- Reach the green goal to advance levels
- Scoring system displayed on screen
- Win condition: Complete all 3 levels

## Gameplay

Navigate the princess through each level, collecting coins and reaching the goal to progress. The levels get progressively more challenging with narrower platforms.
